# 障害報告ガイド

## 障害発生時の手順

1. **即座にサポートデスクへ報告**（優先度: high でチケット作成）
2. 本文に以下を記載する
   - 発生日時
   - 影響範囲（システム名・ユーザー数）
   - 現在の状況
   - 暫定対応状況

## 障害レベル

| レベル | 定義 | 対応 |
|--------|------|------|
| Critical | 全ユーザー影響・データ損失リスク | 即座に全チームアラート |
| High | 一部機能停止・特定ユーザー影響 | 担当チームへ即連絡 |
| Medium | 機能劣化・パフォーマンス低下 | 翌営業日対応 |

## 障害報告テンプレート

```
【障害報告】サービス名：〇〇

■ 発生日時: YYYY-MM-DD HH:mm
■ 影響範囲: 
■ 現象:
■ 原因（判明している場合）:
■ 暫定対応:
■ 恒久対応予定:
```

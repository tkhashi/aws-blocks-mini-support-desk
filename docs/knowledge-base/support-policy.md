# サポート対応ポリシー

## 対応時間

平日 9:00〜18:00（土日祝除く）

## 優先度の定義

| 優先度 | 定義 | 目標初回応答時間 |
|--------|------|----------------|
| high | サービス停止・セキュリティ障害 | 1時間以内 |
| normal | 一般的な問い合わせ・機能要望 | 1営業日以内 |

## チケットのステータス

| ステータス | 説明 |
|-----------|------|
| open | 受付済み・未対応 |
| triage_required | 高優先度チケット確認中 |
| in_progress | 対応中 |
| closed | 解決済み |

## エスカレーション基準

- 48時間以内に解決できない場合はマネージャーへエスカレーション
- セキュリティインシデントは即座にセキュリティチームへ連絡
